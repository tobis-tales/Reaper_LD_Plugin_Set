REAPER LD Plugin Set

Four tools for cue programming in REAPER

Built for lighting designers who write their timecode cues as markers. Each guide is one page — what the plugin is for, how to use it, and the things that are easy to trip over.

The plugins

Live BPM Analyzer Reads the real tempo out of the audio, to two decimals, and sets the project tempo without moving anything you have already placed. MIDI notes to project markers Write your cue rhythm as MIDI and get a marker on every note — named after the note, coloured by pitch. Rename selected markers Builds MA-Tools cue names for a whole set of markers at once, with a live preview and cue numbering that wraps. Copy Markers Copies a block of markers somewhere else, keeping spacing, names and colours.

Before you start

Two REAPER extensions are needed — and both are included. Neither ships with REAPER, so the package carries them for macOS and the installer offers to put the right one in place. Nothing to download.

ReaImGui — required by all four plugins. Without it the Live BPM Analyzer refuses to start and the other three fall back to plain system dialogs.

js_ReaScriptAPI — needed by Rename selected markers to read the manager's list order. Without it the cue numbering silently comes out in a different order.

If you already have either of them, the installer leaves it strictly alone — it never overwrites or downgrades what you have. Updates come from ReaPack, as usual.

Install by running steelblue_install.lua once — in REAPER, under Actions → Show Action List → New action… → Load ReaScript…. It brings the extensions, copies the plugins into REAPER's own Scripts folder, registers all four as actions, and walks you through putting a shortcut on each.

REAPER does not let a script set a key binding — it can only open its own dialog for it. So the installer opens that dialog once per plugin and you press whatever keys suit you. These guides therefore say "the shortcut you assigned" rather than naming keys: yours are the only ones that exist.

Never got round to it? Everything is in Actions → Show Action List under the plugin's file name, and runs from there just as well.

The one thing everybody trips over: before starting a marker plugin from a shortcut, click once into the arrange view. While the Region/Marker Manager has keyboard focus it swallows the shortcut — the window never appears, and your selection is cleared for good measure.

Try it out

The Demo Project folder next to these guides contains a small REAPER project — a beat at exactly 128 BPM, a MIDI item, and five named markers — so you can follow every step here without touching a real show file.
