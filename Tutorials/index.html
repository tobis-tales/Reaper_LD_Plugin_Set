REAPER LD Plugin Set

Four tools for cue programming in REAPER

Built for lighting designers who write their timecode cues as markers. Each guide is one page — what the plugin is for, how to use it, and the things that are easy to trip over.

The plugins

Live BPM Analyzer Reads the real tempo out of the audio, to two decimals, and sets the project tempo without moving anything you have already placed. MIDI notes to project markers Write your cue rhythm as MIDI and get a marker on every note — named after the note, coloured by pitch. Rename selected markers Builds MA-Tools cue names for a whole set of markers at once, with a live preview and cue numbering that wraps. Copy Markers Copies a block of markers somewhere else, keeping spacing, names and colours.

Installing

One script does the whole job. It takes about a minute.

  1. Open the disk image and start the installer

    In REAPER: Actions → Show Action List → New action… → Load ReaScript…, then pick steelblue_install.lua from the disk image and run it.

    That is the only fiddly bit, and it only happens once.

  2. Let it install the two extensions

    REAPER needs ReaImGui for the plugin windows, and js_ReaScriptAPI for the cue numbering in Rename selected markers. Neither ships with REAPER — so both are in the package, and the installer puts the right build for your Mac in place. Nothing to download.

    Already have one of them? It is left strictly alone. The installer never overwrites or downgrades what you have; if ReaPack manages your extensions, ReaPack stays in charge.

  3. Give each plugin a shortcut

    The installer opens REAPER's shortcut dialog once per plugin. Press whatever keys suit you — a script is not allowed to set a key binding, it can only open that dialog, so the choice is genuinely yours.

    Skipping is fine. Everything also runs from Actions → Show Action List under the plugin's file name, and you can add shortcuts there whenever you like.

  4. Quit REAPER, then open it again

    The installer offers to quit for you. REAPER only loads extensions when it starts, so the plugins come alive the moment you reopen it — not before.

    After that the disk image has done its work: the plugins live in REAPER's own Scripts folder now, so you can eject and delete it.

The one thing everybody trips over: before starting a marker plugin from a shortcut, click once into the arrange view. While the Region/Marker Manager has keyboard focus it swallows the shortcut — the window never appears, and your selection is cleared for good measure.

Try it out

The Demo Project folder in the package contains a small REAPER project — a beat at exactly 128 BPM, a MIDI item, and five named markers — so you can follow every step in these guides without touching a real show file.
