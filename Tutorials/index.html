REAPER LD Plugin Set

Four tools for cue programming in REAPER

Built for lighting designers who write their timecode cues as markers. Each guide is one page — what the plugin is for, how to use it, and the things that are easy to trip over.

The plugins

Live BPM Analyzer Reads the real tempo out of the audio, to two decimals, and sets the project tempo without moving anything you have already placed. Cmd + Shift + B MIDI notes to project markers Write your cue rhythm as MIDI and get a marker on every note — named after the note, coloured by pitch. Cmd + Shift + H Rename selected markers Builds MA-Tools cue names for a whole set of markers at once, with a live preview and cue numbering that wraps. Cmd + Shift + N Copy Markers Copies a block of markers somewhere else, keeping spacing, names and colours. Opt + C

Before you start

Two REAPER extensions are needed. Neither ships with REAPER — the easiest way to get both is ReaPack, but a manual install works just as well.

ReaImGui — required by all four plugins. Without it the Live BPM Analyzer refuses to start and the other three fall back to plain system dialogs.

JS_ReaScriptAPI — needed by Rename selected markers to read the manager's list order. Without it the cue numbering silently comes out in a different order.

Install by running steelblue_install.lua once — in REAPER, under Actions → Show Action List → New action… → Load ReaScript…. It checks the extensions, registers all four plugins, and walks you through assigning a shortcut to each.

REAPER does not let a script set a key binding — it only opens its own dialog for it. So the installer proposes a key and you press it. The shortcuts printed in these guides are those proposals; if you picked something else, yours is the one that counts.

The one thing everybody trips over: before starting a marker plugin from a shortcut, click once into the arrange view. While the Region/Marker Manager has keyboard focus it swallows the shortcut — the window never appears, and your selection is cleared for good measure.

Try it out

The Demo Project folder next to these guides contains a small REAPER project — a beat at exactly 128 BPM, a MIDI item, and five named markers — so you can follow every step here without touching a real show file.
