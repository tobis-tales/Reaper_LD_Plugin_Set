Third-party REAPER extensions bundled with the steelblue LD Plugin Set
======================================================================

These two extensions are NOT ours. We ship them unmodified, as a
convenience, so that the plugins work without a separate download.

The installer only ever writes them if the extension is not already
present. It will never overwrite or downgrade one you already have —
if you manage them through ReaPack, ReaPack stays in charge.

WHICH BUILDS ARE IN THIS COPY
-----------------------------

The macOS disk image carries only the macOS builds, and the Windows zip
only the Windows ones — so the folder next to this file has one platform
in it, not both. Whichever you have, the installer picks the build that
matches your machine.

Linux is not bundled. Both projects publish Linux builds at the source
addresses above; the installer can put a file you downloaded yourself
into the right folder.
